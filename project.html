Todo List

The Todo List app is a web-based application that allows users to create and manage their daily tasks. It provides a simple and intuitive interface where users can input their tasks, view the list of tasks, edit existing tasks, and delete tasks.

The app consists of an input field where users can enter their tasks and an "Add" button to add them to the list.

Each task is displayed as a list item with an associated "Edit" button and a "Delete" button.

The "Edit" button allows users to modify the text of a task, while the "Delete" button allows users to remove a task from the list.
